# AWS Real-Time Cyber Threat Detection & Mitigation System

## 🚀 Tech Stack
AWS (EC2, CloudWatch, Lambda, SNS), IAM

## 📌 Project Overview
This project demonstrates a cloud-native system for real-time cyber threat detection and automated mitigation using AWS services.

The system monitors logs and system activity, detects suspicious behavior, and triggers automated alerts and responses.

## ⚙️ AWS Services Used
- Amazon EC2 – monitored instance
- CloudWatch Agent – log and metric collection
- CloudWatch Logs & Metrics – centralized monitoring
- AWS Lambda – automated threat detection
- Amazon SNS – real-time email alerts

## ✨ Key Features
- Real-time log monitoring
- Automated threat detection
- Instant email alerts for critical events
- Serverless and scalable architecture
